# Career Predictor Frontend

React + Vite frontend for the AI Career Prediction application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure Firebase in `src/firebase.js`

3. Create `.env` file:
```env
VITE_BACKEND_URL=http://localhost:8000
```

4. Run development server:
```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deploy to Vercel or any static hosting service.
